# Projeto Calculadora Inteligente
Este é um Projeto referente à construção de uma calculadora inteligente via Python para o exercício do Módulo 5 do curso de Cientista de Dados da EBAC.
O projeto utiliza de estruturas condicionais, de repetição e interações com o usuário, de modo que o mesmo possa ter controle da aplicação.
